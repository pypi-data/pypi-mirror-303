Submit specification for parsing

To upload and parse an NVMe specification, the file needs to be in either DOCX format or the HTML document generated by this tool.

Click the "Specification file..." button, which will open a file selection dialog.

Choose the specification and then click the submit button.

NVMe Spex will attempt to parse the specification, and if successful, a report page will be displayed.


This report page will display linting errors and allow you to inspect figures where linting rules have been violated.

You can then download this report in HTML format, which can be opened in your browser. Additionally, the stage1 (HTML) and stage2 (JSON) generated documents can also be downloaded by clicking the "Download HTML" and "Download JSON" buttons.
